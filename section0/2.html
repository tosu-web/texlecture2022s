Windowsの場合

このページでは Windows の場合の TeX環境設定を紹介します。

Caution

既に環境を整備している場合にやり直す必要はありません。LuaLaTex のセットアップができていなくても、コードの一部を変更すれば支障はありません。 Mac に比べて Windows のセットアップは複雑で面倒です。Mac の PCを持っている方は Macを使うことを強くお勧めします。
また、いくつかの補足情報を前ページに記しているので Windows ユーザーの方も一読お願いします。

Caution

Web ページの作者は Windows での TeX 経験はありますが設定についての知識は乏しいため、 ここでの方法でうまくいかなかった場合は調べてトライした上で不具合箇所と結果を報告いただけると幸いです。

インストール

まずはこのリンクから、install-tl-windows.exe をダウンロードします。

ダウンロードしたらこのファイルを実行し、指示に従ってインストールを進めます。1〜2時間かかります。 途中で「TeXworks をインストール」という項目があったら忘れずにチェックを入れておきましょう！

TeXworksの設定

インストールが完了したら、編集環境として利用できる TeXworks の設定をします。
起動したら「編集」→「設定」と進み、一覧のうちの「LuaLaTeX」を選択して次のように設定します。

  • 名前：LuaLaTeX
  • プログラム：lualatex
  • 引数 (1行ずつ記入すること)
    • -l
    • -ot
    • $synctexoption -guess-input-enc -kanji=utf8
    • $fullname
  • 「実行後、pdfを表示する」にチェック

これで LuaLaTeX を日本語で実行する準備は完了です！

コンパイルの実行

TeX のプログラムを pdf に変換するような翻訳作業をコンパイルといいます。TeXworks でも簡単にコンパイルすることができます。
中身は後で解説するので、とりあえず次のコードをコピーして画像のように書き込み、左上の再生ボタンのようなものの隣のリストで「LuaLaTeX」を選択してからボタンをクリックします。 

\documentclass{ltjsarticle}

\begin{document}
ほげほげ
\end{document}

正しく入力・設定できていれば pdf が表示されるはずです！
